### 여러 메서드를 사용해서 모델이 얼마나 정확한지 정확도를 검증한다.

Support  Vector Machine  

```python
model = svm.SVC(gamma='scale')
model.fit(x_train, y_train)
# model.fit()
# 주어진 데이터로 모델을 학습한다.

## 학습이된 모델을 검증시킨다.

print('SVM : %.2f' %(metrics.accuracy_score(y_pred, y_test) * 100))
```
